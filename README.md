# AI-Based-Early-Plant-Disease-Detection-System-in-Arabic
AI-Based Early Plant Disease Detection System

An advanced AI-powered system for early detection of plant diseases using image processing and deep learning.
The project includes a fully designed Arabic user interface, a Flask-based backend, and a trained CNN model capable of analyzing plant leaf images and generating detailed diagnostic reports.
This repository is intended for educational and research purposes and demonstrates how artificial intelligence can support smart agriculture through accurate, fast, and automated disease detection.
